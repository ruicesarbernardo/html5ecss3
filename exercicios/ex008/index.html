Principais Formatações

Negrito / Destaque

Nesta frase temos um termo em negrito usando a tag <b>, que é uma tag (Não Semântica)

Nesta frase, tenho um termo em destaque usando a tag <strong>(Semântica)!

Itálico / Ênfase

Nesta frase, temos um termo em itálico usando a tag <i>, (Não Semântica)!

Nesta frase, temos um termo em ênfase usando a tag <em> (Semântica)!

Texto Marcado

Podemos criar também um texto marcado usando a tag MARK! (Por defeito a cor do MARK é amarelo, mas podemos mudar essa cor usando um comando CSS (style="") dentro da tag <mark>, mas este comando só altera o mark em questão. Se houver mais e quisermos mudar todos, terá que ser em head / style / mark {} ). Contudo a formatação em head/style/mark só funciona nos mark que não tenham essa formatação aplicada na parte de html.

Texto Grande e Pequeno

Estamos a criar um texto grande e um texto pequeno neste parágrafo. O <big> está a vermelho (no VSC) porque vai ser "descontinuado" em breve, mas o <small> ainda se irá manter!

Texto Apagado

Podemos marcar um texto como texto excluido/apagado, para indicar que ele deve ser lido mas não considerado, com a tag <del>

Texto Inserido

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois, com a tag <ins>.

Existe também, a tag <u> para sublinhado. (Não Semântica)

Texto Sobrescrito

Para inserir coisas do tipo x20+3, usamos a tag <sup>.

Texto Subscrito

Para inserir coisas do tipo H2O, usamos a tag <sub>.
